# Crime-Scale-Prediction-
The aim of the study is to develop a model to predict the crime scale of a new region, knowing its demographic data. 
## [All code can be seen here](https://www.kaggle.com/vassylkorzh/crime-scale-prediction)
### Used tools :
* Python 3
* Scikit-Learn
* Pandas
* NumPy
* Seaborn
* Matplotlib
